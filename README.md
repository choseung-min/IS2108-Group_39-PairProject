# AuroraMart

Contributors:
Cho Seungmin, Jesper Tay Shi Jie

AuroraMart is a full-stack Django web application that replicates the core systems behind a modern e-commerce platform. It includes a customer-facing storefront for browsing products and managing a shopping cart, as well as an administrative interface for managing products, inventory, and customer records through CRUD workflows.

The platform also integrates pre-trained decision tree and association rule models to generate personalised product recommendations and cross-sell suggestions across product pages.

Built with Django (Python) for the backend, HTML/CSS for the frontend, and SQLite for the database.

## Demo
Demo video: https://youtu.be/T-bE96HqsLU

## Features
- Customer storefront for browsing products
- Shopping cart functionality
- Admin interface for managing products, inventory and customer information
- Product recommendations powered by Decision Tree & Association Rules models

## Tech Stack
- Python
- Django
- HTML / CSS
- SQLite
